# Linear Regression Solver

Starter repository for the **Numerical Computing** capstone on [DutchEngineer](https://dutchengineer.org).

## What you're building

A from-scratch linear regression solver implementing both the normal equation and gradient descent. You then validate both implementations against scikit-learn — proving the math, the numerics, and the engineering all hold.

## Getting started

1. Fork this repository
2. Clone your fork: `git clone https://github.com/<your-username>/linear-regression-solver-starter`
3. Follow the capstone instructions in your DutchEngineer dashboard
